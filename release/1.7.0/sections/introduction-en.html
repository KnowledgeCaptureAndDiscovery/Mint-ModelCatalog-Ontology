Introduction back to ToC

This document describes the software description ontology for models (sdm). This ontology extends the [Software Description Ontology](https://w3id.org/okn/o/sd#) (which describes in detail the metadata of a software component, including its inputs, parameters and variables) with additional metadata that is particular to physical and empirical and theory-guided models. For example, sdm introduces concepts for capturing the time intervals associated with models, the physical processes they represent (e.g., infiltration, snow melt, evapotranspiration, etc.) or the details of the different geospatial grids they use, which may have different resolution, dimensions (2D, 3D or point-based) and shapes (triangular, cell, irregular, etc). The ontology also describes how a concrete model setup has been calibrated for a particular region, in order to better understand its obtained results, assumptions and limitations.
